# Summary

Three core prompt engineering patterns—chain-of-thought, system prompts, and few-shot examples—significantly improve LLM output quality and reliability.

# Notes

1. **Chain-of-thought (CoT)**: Elicit step-by-step reasoning with phrases like 'think step by step' or by providing worked examples. Boosts accuracy 20-40% on complex reasoning.

2. **System prompts**: Define persona, constraints, and output format upfront. Reduces repeated instructions and improves consistency across a conversation.

3. **Few-shot examples**: 2-3 concrete input/output pairs help the model match desired format and style. Essential for structured outputs (JSON, tables, code).
